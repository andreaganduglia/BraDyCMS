Template documentation

class publicHtml

publicHtml functions return structured data (strings / arrays). If you are looking for ready to use, well formatted html please check dressHtml documentation

Template: {% raw %} {{ html. }} {% endraw %}
Method Return Example
get(el) (string) Returns element of $data variable {% raw %}{{ html.get('context') }}{% endraw %} 
article
getArticle(el) (string|array) Returns element of article array data, if context is article, or false {% raw %}{{ html.getArticle('title') }}{% endraw %} 
Lorem Ipsum
getLanguages() (array) Returns array of available languages
  • code
  • string
  • is_current
  • href
 {% raw %}{{ html.getLanguages() }}{% endraw %} 
array(
   array(
     'code'=>'it',
     'string'=>'Italiano',
     'is_current'=>true,
     'href'=>'http://bradypus.net/it'
   ),
   array(...)
)
getContext() (string) Returns context: article, section, search or home {% raw %}{{ html.getContext() }}{% endraw %} 
section
getSearchString() (string|false) If context is search returns searched string {% raw %}{{ html.getSearchString() }}{% endraw %} 
bradypus
getSectionName() (string|false) If context is section returns displayed section name.

If context is article returns article section (::Same as getArticle('section'))

 {% raw %}{{ html.getSectionName() }}{% endraw %} 
about_us
getArticleBody() (array|false) Returns article array, if context is article

:: Same as get('article')

   'section' => string,
   'tags' => string,
   'created' => datestamp,
   'publish_on' => datestamp,

getArticlesList(section) (array|false) Returns list of articles.
  • If context is section list of current section articles will be returned
  • If context is search list of found articles will be returned
  • If context is article list of article siblings will be returned (:: same as getArticleSiblings())
  • If section parameter is present lis of section acrticles will be returned
